# 🧠 NexusAI — ذكاء اصطناعي مفتوح المصدر

## 📱 التثبيت على Redmi 12

### الطريقة 1: تثبيت PWA (الأسهل — لا يحتاج متجر)

**الخطوة 1 — ارفع الملفات مجاناً على GitHub Pages:**
1. اذهب إلى [github.com](https://github.com) وأنشئ حساباً
2. أنشئ repository جديد باسم `nexusai`
3. ارفع كل الملفات (index.html, style.css, app.js, sw.js, manifest.json, الأيقونات)
4. اذهب إلى Settings → Pages → Source: main branch → Save

**الخطوة 2 — ثبّت على Redmi 12:**
- افتح Chrome على هاتفك
- اذهب إلى: `https://اسمك.github.io/nexusai`
- انتظر تحميل التطبيق بالكامل
- اضغط على القائمة (⋮) في الزاوية العلوية
- اختر **"إضافة إلى الشاشة الرئيسية"**
- ✅ يظهر على شاشتك كتطبيق أندرويد كامل!

### الطريقة 2: تشغيل مباشر بدون رفع

1. حمّل ملف ZIP وافتحه على هاتفك
2. افتح index.html في متصفح Chrome
3. يعمل فوراً! (ببعض القيود بدون HTTPS)

## 🔑 إعداد مفتاح API (مجاني)

1. اذهب إلى: https://console.anthropic.com
2. أنشئ حساباً مجانياً
3. API Keys → Create Key
4. انسخ المفتاح (يبدأ بـ sk-ant-)
5. في التطبيق: ⚙️ الإعدادات → مفتاح API → الصق → حفظ
